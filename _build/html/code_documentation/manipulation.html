void move_arm(bool gather_hint)

move the robot in one pose

This function allows the robot to move its arm in a certain Pose, depending on whether gather_hint gets true or false

Note

The “home” Pose, refers to the partially extended, initial Pose of the detectiBot’s arm. Otherwise, the “gather_hint” refers to the pose needed for gathering the hint, nearby the marker

Parameters

gather_hint – [in] Boolean variable needed for distinguishing between two differet detectibot’s arm Poses

Returns
